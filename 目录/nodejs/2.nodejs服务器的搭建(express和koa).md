## I.express和koa服务器的搭建

### 1.express

##### A.express myapp（文件名）

##### B.cd myapp

##### C.npm install

##### D.npm start

##### E.在浏览器地址输入：localhost：3000，页面出现express就安装成功了。

### 2.koa

##### A.全局安装：npm install koa-generator -g

##### B.安装项目：koa project(项目名)

##### C.进入创建项目：cd project

##### D.初始化项目包：npm install

##### E.运行项目：npm run dev

##### F.地址栏输入： http://localhost:3000/   出现Hello World Koa! 便是成功了
